x-hide

Это аналог директивы x-show управляет видимостью элемента с помощью свойства display. Она не удаляет элемент из DOM, а лишь скрывает его, устанавливая display: none.

Однако, чтобы x-hide работал, вы должны добавить на страницу следующий CSS-код.

[x-hide] { display: none !important; }
