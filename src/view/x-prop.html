x-prop

Директива позволяет связать значения полей ввода input, select и textarea к x-data без объявления их в директиве. Такой подход позволяет не повторяться и сделать html представление более лаконичным. Сравните 2 по смыслу идентичных подхода: 

<form x-data>
  <input type="text" value="Hello" x-prop="test">
  <span x-text="test"></span>
</form>
 
<form x-data="{test: 'Hello'}">
  <input type="text" value="Hello" @keydown.delay.1ms="test = $el.value">
  <span x-text="test"></span>
</form>

Во втором случае не требуется объявления в директиве x-data и события ввода.

Text inputs

<form x-data>
  <input x-prop="message" type="text">
  <span x-text="message"></span>
</form>

Textarea inputs

<form x-data>
  <textarea x-prop="message"></textarea>
  <span x-text="message"></span>
</form>

Checkbox inputs

Одиночный checkbox с логическим значением

<form x-data>
  <input type="checkbox" id="checkbox" x-prop="show">
  <label for="checkbox" x-text="show"></label>
</form>

Несколько checkbox, привязанных к массиву

<form x-data>
  <input type="checkbox" value="red" x-prop="colors">
  <input type="checkbox" value="orange" x-prop="colors">
  <input type="checkbox" value="yellow" x-prop="colors">
  Colors: <span x-text="colors"></span>
</form>

Модификаторы

.local и .cookie

Позволяют автоматически сохранять значение привязанного свойства в куках или локальном хранилище и восстанавливать его при последующих загрузках страницы. Это удобно для хранения пользовательских предпочтений, например, темы оформления, имени пользователя или других параметров.

Сохранение значений формы

<form x-data>
  <div>Сохранение в local storage</div>
  <input type="text" x-prop.local="local">
  <div x-text="local"></div>
  <br>
  <div>Сохранение в куках</div>
  <input type="text" x-prop.cookie="cookie">
  <div x-text="cookie"></div>
</form>
