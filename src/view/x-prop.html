x-prop

Директива позволяет связать значения полей ввода input, select и textarea к x-data без объявления их в директиве. Такой подход позволяет не повторяться и сделать html представление более лаконичным. Сравните 2 по смыслу идентичных подхода: 

<form x-data>
  <input type="text" value="Hello" x-prop="test">
  <span x-text="test"></span>
</form>
 
<form x-data="{test: 'Hello'}">
  <input type="text" value="Hello" @keydown.delay.1ms="test = $el.value">
  <span x-text="test"></span>
</form>

Во втором случае не требуется объявления в директиве x-data
