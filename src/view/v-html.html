v-html

Директива v-html позволяет вставлять HTML-код в элемент, заменяя его текущее содержимое. Это аналог свойства innerHTML в JavaScript.

⚠️ Опасность XSS

Используйте только для доверенного контента и никогда - для контента, предоставленного пользователем. Динамический рендеринг HTML третьими лицами может легко привести к уязвимостям XSS.

Вот простой пример использования v-html для отображения имени пользователя:

<div v-data="{ username: '<strong>Yan Aleksandrov</strong>' }">
  Имя: <span v-html="username"></span>
</div>
