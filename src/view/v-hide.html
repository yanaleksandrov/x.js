v-show

Это аналог директивы v-show управляет видимостью элемента с помощью свойства display. Она не удаляет элемент из DOM, а лишь скрывает его, устанавливая display: none.

Однако, чтобы v-show работал, вы должны добавить на страницу следующий CSS-код.

[v-show] { display: none !important; }
