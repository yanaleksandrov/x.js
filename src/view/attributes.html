Атрибуты

Youla.js предоставляет простой способ динамически управлять атрибутами HTML-элементов на основе текущего состояния данных. Синтаксис реализуется через добавление двоеточия перед названием атрибута. 

<tag :атрибут="выражение"></tag>

Вот простой пример реализации смены цвета кнопки: 

<div class="demo" v-data="{isRed: false}">
  <button @click="isRed = !isRed" :style="isRed && {color: 'red'}">Toggle Color</button>
</div>

Синтаксис : может применяться ко всем стандартным HTML-атрибутам, включая текстовые (href, src, title, placeholder), булевые (disabled, checked, readonly, required), объектные (class, style) и пользовательские (data-*, aria-*).
