x-data

В X.js всё начинается с директивы x-data. x-data определяет фрагмент HTML как новый компонент и предоставляет реактивные данные для ссылки на этот компонент.

Модификаторы

.local и .cookie

Позволяют автоматически сохранять значение привязанного свойства в куках или локальном хранилище и восстанавливать его при последующих загрузках страницы. Это удобно для хранения пользовательских предпочтений, например, темы оформления, имени пользователя или других параметров.

Принять всплывающее окно с файлами cookie

Окно "Мы используем файлы cookie!" будет появляться при каждой перезагрузке страницы. Однако после нажатия кнопки "Ок, принято" x.js запомнит выбор пользователя и не будет отображать его.
