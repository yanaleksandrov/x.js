x-show

Директива x-show управляет видимостью элемента с помощью свойства display. Она не удаляет элемент из DOM, а лишь скрывает его, устанавливая display: none. 

<div x-data="{ open: false }">
  <button @click="open = ! open">Переключить</button>

  <div x-show="open">
    Контент...
  </div>
</div>

При нажатии кнопки "Переключить" выпадающий список будет отображаться и скрываться соответствующим образом.

Контент...

Если состояние x-show при загрузке страницы "по умолчанию" равно "false", вы можете захотеть использовать директиву x-hide, чтобы избежать "мерцания страницы" (эффекта, который возникает, когда браузер отображает ваш контент до того, как X.js завершит его инициализацию и скрытие).

x-hide - это обратный аналог x-show, в котором стартовое состояние - скрыто.
